## Installation

Install my-project with npm

**Frontend**

```bash
  cd frontend
  npm install 
```

## Run Locally

Clone the project

```bash
  git clone git@github.com:GautamGunecha/thehousemonk.git
```

Go to the project directory

**Frontend**

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
